# RemapXiaomiMi9PowerButton

Just remapping the IA buttor for the POWER button

Tested and working on the EvolutionX android 15 custom rom with KernelSuNext

You can Download the zip from the realeases (if I remember to put it there) and just install as a module on KernelSuNext
Or you can git clone this repository and create a zip with all the files in it (not the .git folder tho) and just flash it!
